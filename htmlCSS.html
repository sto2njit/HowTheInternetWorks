HTML and CSS

Hypertext Markup Language (HTML) is the computer software language used by developers to create and edit webpages. This allows developers to manipulate text by font and placement on the screen, as well as adding content such as images and videos to the page. Cascading Style Sheets (CSS) takes the text content from HTML and edits it into different colors, designs and fonts to make the webpage more visibly aesthetic.

W3C

The World Wide Web Consortium (W3C) is an organization that advocates for best practices and sustainability protocols for the internet. This organization was started, in part, by the inventor of the world wide web: Sir Tim Berners-Lee. The W3C calls for an open and free internet for users all around the world to access and utilize without interruption from governments, corporations or entities that would abridge these capabilities of the web.

Tim Berners-Lee was the first to send HTTP protocols to another server, effectively creating the world wide web (image retrieved from Wikipedia.)
