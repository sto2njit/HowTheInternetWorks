HTTP

When your computer device connects with the internet by sending or receiving information, this is communicated in the language of Hypertext Transfer Protocols ("HTTP.") Essentially, these protocols perform either "get" or "post" requests. The get requests are used when the user is trying to use their web browser and access a domain, whereas the post request is when the user sends information (like search requests for example) back to the server.

Secure Web Usage

Without proper security, webpages may be susceptible to internet "hackers" stealing data and information. However, if you are on a website with the HTTPS protocols (Hypertext Transfer Protocols Secure) then the website utilizes Transport Layer Security that verifies the safety of the website for information transfer. It is an extra layer of protection so that information including passwords, social security numbers and financial information is secure if transferred on the web.

The lock icon on your URL will indicate that the website you are accessing uses secure protocols.
