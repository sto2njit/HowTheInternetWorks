DNS Servers

When you enter the domain for a website or webpage, the information is sent to a DNS server that will send your device a string of numbers serving as the address for accessing that domain. There are several DNS servers that utilize specific protocols for different types of websites, including ".com", ".org", and ".gov" just to name a few. If the DNS server is not aware of the address of a given domain name, it will reference other DNS servers to find such an address.

Information retrieval is sent from one DNS server and back again to procure the address information to the user.
